Foire aux questions (FAQ)

1. Pourquoi mes données ont-elles disparu après un changement d’appareil ?

TripWander utilise un stockage local. Avant de changer d’appareil, veuillez exporter vos données de voyage puis les importer sur le nouvel appareil.

2. Dois-je payer à nouveau après une réinstallation de l’application ?

Non. Les fonctionnalités premium correspondent à un achat unique. Elles peuvent être restaurées gratuitement en utilisant le même identifiant Apple.

3. Je ne trouve pas les images exportées dans l’application Photos.

Veuillez vérifier que l’application dispose de l’autorisation d’accès aux photos. Les images exportées sont généralement enregistrées dans l’album « Récents ».

4. Comment supprimer une étape de voyage ?

Dans la vue chronologique du voyage, faites glisser l’élément concerné vers la gauche pour le supprimer.
